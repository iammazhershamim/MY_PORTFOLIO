CSS Units — Short Demo

नीचे हर सेक्शन में box को resize करके observe करो — mobile/desktop दोनों पर behavior अलग दिखेगा।

Percentage (%) — relative to parent
Parent (width: 80% of body)
Child (width: 50% of parent)
Observe: जब आप browser की width घटाओगे तो parent और child दोनों proportional छोटे होंगे — क्योंकि % parent के हिसाब से है.
Pixels (px) — fixed size
300px wide — fixed
Observe: px fixed रहता है; छोटे screen पर overflow या horizontal scroll आ सकता है.
Viewport units (vh / vw)
Hero height = 40vh
Child width = 50vh
Observe: 50vh का मतलब viewport height का 50% — parent irrelevant. Resize window height to see effect.
rem vs em — font relative
This text = 1rem (16px)
Parent font-size = 2rem (32px)
Child font-size = 1.5em → 1.5 * 32px = 48px
Observe: rem is relative to root (html), em relative to parent element's font-size.
vmin / vmax / ch
Height = 20vmin (20% of smaller viewport dimension)
Width = 30ch (approx width for 30 characters of this font)
vmin uses smaller of vw/vh; ch depends on glyph width — useful for text containers.

Tip: Responsive layouts → prefer % (for layout) + rem for typography + clamp()/media-queries for fine control. Use vh for full-screen sections (hero), but beware mobile browser UI changing viewport height.
